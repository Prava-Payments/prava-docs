## Overview

The `deleteIntent()` method cancels an intent permanently. The underlying card network mandate is revoked — it can no longer be used to issue tokens or process transactions. Requires passkey authentication.

<Warning>
This action is **irreversible**. Once deleted, the intent can never be invoked again.
</Warning>

## Method Signature

```typescript
prava.deleteIntent(params: { intentId: string }): Promise<DeleteIntentResult>
```

## Parameters

<ParamField path="intentId" type="string" required>
  The intent to cancel
</ParamField>

## Return Value

<ResponseField name="result" type="DeleteIntentResult">
  <Expandable title="properties">
    <ResponseField name="intentId" type="string">
      The cancelled intent ID
    </ResponseField>

    <ResponseField name="status" type="string">
      Always `'cancelled'`
    </ResponseField>
  </Expandable>
</ResponseField>

## Example

<CodeGroup>
```typescript AI Agent
import { PravaSDK } from '@prava-sdk/core';

const prava = new PravaSDK({ publishableKey: 'pk_live_xxx' });

await prava.deleteIntent({
  intentId: 'int_m7kx9...',
});
// Passkey prompt → mandate cancelled with card network
// This intent can never be invoked again
```

```typescript React Component
function IntentManager({ intent, onDeleted }) {
  const [deleting, setDeleting] = useState(false);
  const prava = new PravaSDK({ publishableKey: 'pk_live_xxx' });

  async function handleDelete() {
    if (!confirm(`Cancel authorization for ${intent.merchant}?`)) return;

    setDeleting(true);
    try {
      await prava.deleteIntent({ intentId: intent.intentId });
      onDeleted(intent.intentId);
    } catch (error) {
      if (error.code === 'PASSKEY_REJECTED') {
        alert('Cancellation declined');
      } else {
        alert(`Failed to cancel: ${error.message}`);
      }
    } finally {
      setDeleting(false);
    }
  }

  return (
    <div>
      <span>{intent.merchant} — ${intent.amount}</span>
      <button onClick={handleDelete} disabled={deleting}>
        {deleting ? 'Cancelling...' : 'Cancel'}
      </button>
    </div>
  );
}
```
</CodeGroup>

## Error Handling

| Code | Cause | Resolution |
|------|-------|------------|
| `PASSKEY_REJECTED` | User declined the passkey prompt | Allow retry |
| `PASSKEY_UNAVAILABLE` | Device doesn't support WebAuthn | Inform user |
| `INTENT_EXPIRED` | Intent already expired | No action needed |
